jogo criado com base nas aulas da @gabriela-pinheiro - plataforma DIO (Digital Innovation One).

alterações realizadas:

- cor cabeça;

- comidinha não nasce "dentro" da cobrinha;
- pontuacao;
- restart sem F5 na página.
